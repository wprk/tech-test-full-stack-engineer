## Description

This is the HiPages Tech Test - Full Stack Engineer Server. It uses a standard [Nest](https://github.com/nestjs/nest) framework boilerplate.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
